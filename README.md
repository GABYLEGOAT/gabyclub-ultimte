# ⚡ GABY Club Ultimate - Mode Invincible

Bienvenue sur le dépôt officiel du **GABY Club** ! Ce projet est un mini-jeu d'arcade (Pong) avec un mode défi "IA Invincible" et un espace pour soutenir le club.

## 🚀 Fonctionnalités
* **Pong Arcade :** Un moteur de jeu fluide en HTML5 Canvas.
* **IA Imbattable :** Un adversaire robotique qui ne rate (presque) jamais.
* **Système de Score :** Sauvegarde de ton record personnel sur ton navigateur.
* **Thème Dynamique :** Mode sombre/clair pour jouer confortablement.
* **Espace Fan :** Compteur de clics pour encourager le club et bouton de soutien PayPal.

## 🛠️ Installation
Pas besoin d'installation complexe ! 
1. Télécharge les fichiers du dépôt.
2. Ouvre le fichier `index.html` dans n'importe quel navigateur (Chrome, Firefox, Safari).
